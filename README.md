# parser

## Запуск приложения
```bash
#для запуска или пересборки контейнеров
make rebuild 
# или что бы поднять уже собранные контейнеры 
make up 
```

- Запускается вместе с базой данных (MariaDB)
- Приложение доступно по адресу [http://localhost:8080](http://localhost:8080)
